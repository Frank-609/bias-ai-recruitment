What are we doing:
------------------------

We are building a controlled experimental pipeline to study bias in large language models 
when they act as recruiters. We generate realistic synthetic CVs, then create matched 
variants where only one attribute (such as gender, name origin, age, or career gap) is changed 
while everything else remains as identical as possible. These CVs are then presented to 
different LLMs (e.g., ChatGPT, Mistral), which are asked to evaluate candidates and choose 
who to hire. By structuring the data this way, we isolate the effect of a single variable and 
ensure that any difference in model decisions can be attributed to that factor rather than 
unrelated differences in qualifications.

Our goal is to measure and analyze systematic patterns in these decisions to identify potential 
biases. We aggregate results across many controlled comparisons and examine metrics such as 
selection rates and score differences for each attribute. This allows us to compare models, 
prompts, and conditions, and ultimately understand how and when biases emerge. 
The project is therefore not just about generating CVs, but about creating a reproducible 
experimental framework that produces meaningful, interpretable evidence about bias in AI-driven 
hiring scenarios.


For the setup:
--------------
Creating venv and install dependencies:
---------------------------------------
python -m venv .venv
.venv\Scripts\activate        # Windows
# OR
source .venv/bin/activate     # Linux/macOS

pip install -r requirements.txt

API Key Setup: Create .env in the root folder and type inside it:
-----------------------------------------------------------------
OPENAI_API_KEY=apikeyforOPENAI
MISTRAL_API_KEY=apikeyforMISTRAL

Main Commands:
--------------
Running everything: CV generation, Variant creation, Session building, Evaluation, Analysis
-------------------------------------------------------------------------------------------
python -m src.run_pipeline \
  --job "Data Analyst" \
  --languages English \
  --template-count 2 \
  --provider openai \
  --model gpt-4o-mini \
  --scenario single_attribute \
  --prompts neutral_v1 \
  --replicates 1 \
  --concurrency 2 \
  --reset

Only CV generation:
-------------------
python -m src.generator templates \
  --job "Data Analyst" \
  --language English \
  --count 10 \
  --provider openai \
  --model gpt-4o-mini \
  --reset

then:

python -m src.generator variants \
  --provider openai \
  --model gpt-4o-mini \
  --scenario single_attribute \
  --reset


For previewing a CV:
--------------------
a specific one: find its id in jsonl files (example: cv_2f2bbfa37ce0)
python -m src.preview --cv-id cv_2f2bbfa37ce0
or simply just pick a random one
python -m src.preview --random
